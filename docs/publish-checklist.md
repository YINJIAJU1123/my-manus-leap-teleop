# Publish Checklist

Recommended follow-up items for the public repository:

1. Add a real project license if you want others to reuse the code.
2. Add a short demo video or GIF to the repository front page.
3. Add GitHub topics such as `ros2`, `teleoperation`, `robot-hand`, `manus`, `leap-hand`.
4. Add one architecture figure showing `MANUS SDK -> ZMQ -> glove -> telekinesis -> LEAP`.
5. If you later want a private all-in-one version, keep that as a separate private repository.
